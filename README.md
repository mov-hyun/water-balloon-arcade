# Water Balloon Arcade

HTML, CSS, JavaScript만으로 만든 단일 파일 브라우저 아케이드 게임입니다.

물풍선을 설치해 블록을 부수고, 아이템을 모으고, 제한 시간 안에 AI 상대를 물방울에 가두는 게임입니다.

## 주요 기능

- 키보드와 터치 조작 지원
- 물풍선 설치와 십자형 물줄기 폭발
- 부술 수 있는 블록과 아이템 드롭
- AI 상대
- 세 가지 화면 테마
- 세 가지 맵 크기

## 실행

Open `index.html` in a browser.

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## English

A small single-file browser arcade game built with HTML, CSS, and JavaScript.

Place water balloons, break blocks, collect items, and trap AI opponents before time runs out.
